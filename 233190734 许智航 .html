Programming languages

There are so many programming languages around now. Most have advantages

and disadvantages,and each has a particular style of programming which is

used. Here is a table showing a few languages:

Languages Usual programming style Popularity
C Procedural Very popular
Haskell Functional Medium
Java Objective-oriented Very popular
Python Multi-paradigm Very popular
R Multi-paradigm Very popular

There are also some amusing novelty languages such as LOLCODE which is not taught at the University of Auckland
